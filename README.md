## 🚀 Deploy to GitHub Pages

1. Push the entire folder contents to a repo named `<your-username>.github.io`
2. Go to **Settings → Pages → Source → Deploy from branch → main / root**
3. Your site will be live at `https://<your-username>.github.io`

## 🛠 Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, animations, responsive layout
- **JavaScript (ES6+)** — Vanilla JS with no build step required
- **Three.js r128** — 3D WebGL background (loaded via CDN)
- **Google Fonts** — Cormorant Garamond, DM Mono, Bebas Neue
